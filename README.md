# XAML-to-SVG

Convert XAML contents from MS Expression Design into nice, usable, sharable SVGs

Export from Design with Edit > Copy XAML (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd>)
